PAYER AVEC BRIDGE, SOYEZ RASSURÉ

Bridge est l'application qui sécurise les transactions entre particumiers qui commandent sur les sites de petites annonces, du paiement à la livraison, gratuitement.

Comment ça marche ?

Pour les échanges entre particuliers, que vous soyez acheteur ou vendeur

Vous êtes acheteur ?

  • J'hachète un produit
  • Je paye avec Bridge, mon argent est bloqué
  • Je reçois mon colis
  • L’argent est débloqué pour le vendeur

Vous êtes vendeur ?

  • Je vends un produit
  • Je reçois l’argent sur Bridge, l’argent est bloqué
  • J’envoie la commande
  • L’acheteur reçoit la commande, l’argent est débloqué
